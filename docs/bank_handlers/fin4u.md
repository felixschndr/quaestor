# fin4u retirement plan

Retirement provisioning via the Alte Leipziger [fin4u portal](https://www.alte-leipziger.de/service/rund-um-ihre-vertraege/kundenportal-fin4u).

Quaestor connects **directly** to the portal (your credentials are only ever exchanged between your Quaestor server and the provider, no third party is involved).

There is no support for individual transactions as the portal does not provide this information; only the balance is fetched.
