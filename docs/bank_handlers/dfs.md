# DFS retirement plan

Retirement provisioning of the Deutsche Flugsicherung GmbH
([dfs-vorsorgeplan.de](http://www.dfs-vorsorgeplan.de) / [value-account.eu](https://www.value-account.eu)).

Quaestor connects **directly** to the provider (your credentials are only ever exchanged between
your Quaestor server and the provider, no third party is involved).
